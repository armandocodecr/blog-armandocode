El efecto de escritura en las páginas web es uno de los más conocidos y estéticos. 

Es un efecto muy útil para Portfolios, o para resaltar cierto tipo de información en tus páginas web. Pero eso sí, no abuses de este efecto
en una misma página, ya que puede acabar arruinando en parte el diseño. 🙅‍♂️

Para el **HTML** ocuparíamos lo siguiente. 👇

```html
<div class="text">
    <h1>Tu texto preferido (ArmandoCode)</h1>
</div>
```

Y para la parte del **CSS** necesitaríamos esto. 🖌

```css
.text{
    display: inline-block;
}

.text h1{

    font-size: 4rem;
    color: black;
    overflow: hidden;
    border-right: .20em solid rgb(0, 128, 248);
    white-space: nowrap;
    animation: typing 1.5s steps(20, end) forwards, 
    blinking 1s infinite;

}

@keyframes typing {
    from {width: 0;}
    to {width: 100%;}
}

@keyframes blinking {
    from {border-color: transparent;}
    to {border-color: rgb(0, 128, 248);}
}
```

Y con esto ya tendríamos el efecto listo para ser utilizado ✨

Estás en la libertad de extender los estilos de este, y hacer de este efecto todavía más estético.
